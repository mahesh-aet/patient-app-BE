# patient-app-BE

### Patient Management API

 - A RESTful backend application built with Spring Boot, PostgreSQL, and Spring Data JPA to manage patient information.

 - The API provides CRUD operations for patient records and exposes documented endpoints using OpenAPI.
````
(http://localhost:8080/swagger-ui/index.html)
````
### How to run

 - Clone from the github repo
 - navigate to project
 - run the application using this command
````
   mvn spring-boot:run
````
 - Application will start at
````
    http://localhost:8080
````
